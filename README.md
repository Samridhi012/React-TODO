# React-Todo-App
Todo list app project using react hooks

<h3>Functionalities:</h3>
<ol>
  <li> Add Todo by clicking add button</li>
  <li> Add Todo by pressing Enter key</li>
  <li> Delete Todo through strikethrough</li>
 </ol>
